## 🚒 프로젝트 배경
✔️ 최근 소방본부는 소방안전 데이터를 분석하여 적극적이고 체계적인 화재 대응 체계 구축 기반을 마련하기 위해 노력하고 있는 추세이며, 소방 안전에 대한 중요성이 점점 더 강조되고 있음을 느끼며, 소방 관련 데이터를 수집·가공하여 빅데이터 분석을 통해 화재 발생 패턴을 분석하고, 공공 데이터의 활용성을 높이는 데 기여하고자 함.  
  
✔️ 국내 소방청 DB를 활용하여 화재 통계 분석, 최근 화재현황, 화재 예방 정보를 제공하고 소통할 수 있는 게시판을 포함한 웹 플랫폼을 구축하여 소방 행정의 효율성을 높이고 국민의 안전을 도모하고자 함.

## 🚒 프로젝트 핵심기능  
1. 전체 화재 통계
2. 최근 화재현황 (한달 단위)
3. 화재 알림 시스템
4. 화재 예방법
5. 알림마당 게시판 (공지사항 & 건의사항)
6. 회원관리
7. 관리자 페이지
